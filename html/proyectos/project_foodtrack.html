FoodTrack. Control de comidas

Para el desarrollo de este proyecto, elaboré una aplicación para el control de comidas y cálculo de calorías y nutrientes.
Para ello, elaboré el frontend con Angular y el backend con Springboot, consumiendo de una API pública para obtener los datos de las comidas.

Mi trabajo en el proyecto: Realicé todos los aspectos que componen la aplicación, desde el diseño de las vistas con Figma, el desarrollo del frontend en Angular y el desarrollo del backend con Springboot y PostgreSQL. Además, se realizó una 'dockerización' del proyecto.
