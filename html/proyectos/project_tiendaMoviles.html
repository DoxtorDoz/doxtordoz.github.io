Tienda de móviles

Para el desarrollo de este proyecto, desarrollé junto a un equipo una página web para una tienda de móviles de nuestra ciudad.
Para ello, se trabajó con HTML, CSS y JavaScript, y utilizando diversas herramientas para el desarrollo de la misma.

Mi trabajo en el proyecto: En este proyecto, me encargué del diseño de las vistas, el diseño, desarrollo e implementación de la base de datos, y en parte del desarrollo de la lógica de la misma.

Tecnologías y herramientas empleadas:

HTML
Javascript
Visual Paradigm
MySQL
Css
Visual Studio Code
PhpMyAdmin
