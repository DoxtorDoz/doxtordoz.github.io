Periódico digital

Para el desarrollo de este proyecto, desarrollé un periódico digital con diversos roles, pudiendo realizar una serie de acciones, como visualizar noticias, publicar noticias, realizar calificaciones y comentarios, y administrar el contenido de la plataforma.

Mi trabajo en el proyecto: Realicé todos los aspectos que componen la aplicación, desde el diseño y desarrollo de los distintos diagramas con Visual Paradigm (Diagrama de Casos de Uso, Diagrama de clases, Diagrama de Bases de Datos, Diagrama de Secuencias, diseño de las vistas), como el desarrollo del frontend (Vaadin) y backend (Java) de la aplicación.

Tecnologías y herramientas empleadas:

Vaadin
Java
Visual Paradigm
MySQL
Eclipse
PhpMyAdmin
HTML
CSS
